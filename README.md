# chat-application

A chat application is a software that allows people to communicate with each other in real-time over the internet. Users can send and receive text messages, share images, videos.

## Front-end

To run frontend

cd frontend

npm i

npm start

## Server or backend

To run server or backend

cd server

npm i

npm run dev
